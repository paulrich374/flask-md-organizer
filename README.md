# Flask Markdown Organizer

## Project Documentation
This project is designed as a web application to manage and organize Markdown files effectively using the Flask framework.

## Features
- **Organize Markdown files**: Easily organize Markdown files into different categories.
- **Search functionality**: Quickly find Markdown files through a powerful search feature.
- **User-friendly interface**: A clean and intuitive UI for easy navigation.
- **Customizable settings**: Configuration options tailored to user preferences.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/paulrich374/flask-md-organizer.git
   cd flask-md-organizer
   ```
2. Create a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```
3. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage Guide
To start the application, run:
```bash
flask run
```
Then navigate to `http://127.0.0.1:5000` in your web browser to access the application.
